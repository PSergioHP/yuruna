# Build container images and push to registry

These scripts will use configuration information from file `config/workloads.yml`.

## Build images locally

- Build containers images locally executing from the folder `automation`.

```shell
build-src.ps1
```

## Pushing container images to registry

- Push containers images to registry executing from the folder `automation`.

```shell
registry-push.ps1
```
